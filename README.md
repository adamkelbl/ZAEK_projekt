# Determinanty cen ojetých vozů Porsche 911
- Závěrečný projekt do předmětu **Základy ekonometrie** (3. semestr Bc. studia) <br>
- Hlavním cílem bylo určit faktory které ovlivňují **cenu ojetého vozu Porche 911** <br>
- Na datech z internetového bazaru [elferspot.com](https://www.elferspot.com/en/) byla provedena **vícenásobná lineární regrese pomocí metody OLS** <br>
- Proměnné zahrnují sérii vozu (911–992), specifikaci (RS, GT, Turbo), typ karoserie, převodovku, nájezd apod.
- Finální model v podobě log-lin s robustními standardními chybami vysvětluje ~ 86 % variability ceny
- Specifikace RS zvyšuje cenu o ~ 52 %, automatická převodovka cenu snižuje o ~ 18 %, barva exteriéru a pohon všech kol se ukázaly jako statisticky nevýznamné
